# 🧠 AI Knowledge Assistant using RAG

An AI-powered assistant that supports:

- PDF Upload
- DOCX Upload
- PPTX Upload
- OCR Image Reading
- Voice Questions
- AI Chat
- Study Notes Generation
- RAG-based Retrieval

## Technologies Used

- Python
- Streamlit
- ChromaDB
- SentenceTransformers
- OpenRouter API
- OCR
- NLP

## Run Project

```bash
streamlit run gemini_rag_app.py
